# CONSERVAR SOSTENIBLE

## Proyecto de Turismo Sostenible

### Descripción
Sitio web para promover el turismo responsable y sostenible, con carrusel automático de imágenes de destinos turísticos.

### Características
-  Carrusel de imágenes automático
-  Rotación cada 5 segundos
-  Navegación responsive
-  Sistema de login/registro
-  Gestión de destinos turísticos
-  Formulario de contacto

### Tecnologías
- Node.js + Express
- PostgreSQL (base de datos)
- HTML5, CSS3, JavaScript
- Swiper.js para carruseles
- JWT para autenticación

### Instalación
```bash
1. Clonar repositorio
2. npm install
3. Configurar .env
4. npm run setup
5. npm run seed
6. npm run dev
